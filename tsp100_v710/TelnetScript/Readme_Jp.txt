スクリプトを使用することで、TSP100LANの#9100ポートの設定を容易に変更する
ことが出来ます。
以下スクリプトの使用方法について説明します。


=========
1. 対応OS
=========

本スクリプトは、以下のOSをサポートしております。

  ・Windows
      Windows 7   (32bit/64bit)
      Windows 8   (32bit/64bit)
      Windows 8.1 (32bit/64bit)
      Windows 10  (32bit/64bit)

  ・Linux
      Red Hat Enterprise Linux 6.7
      openSUSE 13.1
      Fedora 20
      ubuntu 16.04 LTS
      CentOS 6.8

  ・Mac
      Mac OS X 10.6
      Mac OS X 10.7
      Mac OS X 10.8
      Mac OS X 10.9
      Mac OS X 10.10
      Mac OS X 10.11


==========
2. Windows
==========

2-1. Telnetコマンドを使用するため、事前に以下の内容をご確認して下さい。

  ・Windows7
      1). スタートメニューから、「設定」→「コントロールパネル」を選択します。
      2).「プログラムと機能」を選択します。
      3). タスク一覧より、"Windowsの機能の有効化または無効化"を選択します。
      4). ユーザアカウント制御画面では「続行」を選択します。
      5).「Telnetクライアント」のチェックがついていない場合は、チェックを付けて
         「OK」を押下します。

  ・Windows8/8.1
      1).「デスクトップ」画面で「チャームバー」→「設定」→「コントロールパネル」
          を選択します。
      2).「プログラム」を選択します。
      3).「プログラムと機能」より、"Windowsの機能の有効化または無効化"を選択します。
      4). ユーザアカウント制御画面では「続行」を選択します。
      5).「Telnetクライアント」のチェックがついていない場合は、チェックを付けて
         「OK」を押下します。

  ・Windows10
      1). スタートメニューから、「すべてのアプリ」→「Windowsシステムツール」→
          「コントロールパネル」を選択します。
      2).「プログラム」を選択します。
      3).「プログラムと機能」より、"Windowsの機能の有効化または無効化"を選択します。
      4). ユーザアカウント制御画面では「続行」を選択します。
      5).「Telnetクライアント」のチェックがついていない場合は、チェックを付けて
         「OK」を押下します。


2-2. IPアドレスとパスワードの設定

    batファイル内の IPアドレス、パスワードを適切な値に変更して下さい。
    デフォルト値は、以下の通り。
        IPアドレス: "192.168.123.45"
        パスワード: "public"

      1). #9100マルチセッションを無効にする
            "Execute_9100Single.bat" を変更。

      2). #9100マルチセッションを有効にする
            "Execute_9100Multi.bat"  を変更。

    ※パスワードの工場出荷時の値は、"public"です。


2-3. batファイルの実行

    batファイルをダブルクリックして実行して下さい。

      1). #9100マルチセッションを無効にする
            "Execute_9100Single.bat" を実行。

      2). #9100マルチセッションを有効にする
            "Execute_9100Multi.bat"  を実行。


==============
3. Linux / Mac
==============

3-1. Linux環境ではスクリプトの実行のため、以下のパッケージをインストール
     する必要があります。

  ・必要なパッケージ
      - expect
      - telnet

  ・インストール方法
      ターミナル(コンソール)を起動します。
      以下のコマンドを、管理者権限で実行して下さい。
      ※コマンドの実行は、インターネットに接続されている必要があります。

      - Red Hat Enterprise Linux6.7/CentOS6.8
          # yum install expect
          # yum install telnet

      - Fedora20
          # yum install expect

      - openSUSE13.1
          # zypper in expect

      - ubuntu16.04 LTS
          $ suto apt-get install expect


3-2. スクリプトの実行

    ターミナル(コンソール)を起動します。
     第一引数にIPアドレスを設定し、第二引数にパスワードを設定して、スクリプト
    を実行して下さい。

      1). #9100マルチセッションを無効にする

         例)
          /bin/bash 9100Single.sh "192.168.123.45" "public"

      2). #9100マルチセッションを有効にする

         例)
          /bin/bash 9100Multi.sh "192.168.123.45" "public"

    ※パスワードの工場出荷時の値は、"public"です。
